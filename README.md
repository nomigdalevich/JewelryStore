# Jewelry Store

React e-commerce application for browsing, searching, and filtering jewelry products. Includes a shopping cart and an admin interface for managing products.

## Features
- Product search and filtering
- Shopping cart management
- Admin interface for product updates

## Technologies
React, JSX

## How to Run
```bash
# Clone the repository
git clone https://github.com/nomigdalevich/JewelryStore.git

# Navigate to the project folder
cd JewelryStore

# Install dependencies
npm install

# Start the application
npm start
```

Open http://localhost:3000
 in your browser to view the app
